# 스도쿠 게임

스도쿠는 가로, 세로, 그리고 3x3 작은 정사각형 안에서
1부터 9까지의 숫자를 중복되지 않게 채워 넣는 논리 퍼즐입니다.

## 규칙
- 각 행에는 1~9가 중복 없이 들어가야 합니다.
- 각 열에는 1~9가 중복 없이 들어가야 합니다.
- 각 3x3 박스에도 1~9가 중복 없이 들어가야 합니다.

## 함수 설명
- initialize_board: 빈 스도쿠 보드를 생성합니다.
- print_board: 현재 스도쿠 보드를 출력합니다.
- solve_sudoku: 스도쿠 퍼즐을 해결합니다.
- generate_sudoku: 랜덤 스도쿠 퍼즐을 생성합니다.
- play_sudoku: 사용자가 플레이할 수 있도록 게임을 실행합니다.
